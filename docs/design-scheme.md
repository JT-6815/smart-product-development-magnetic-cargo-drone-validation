# 方案设计

## 平台方案
本项目采用 F450 四旋翼套件作为基础平台。选择该方案的原因是零件通用、配件易获取、结构清晰，适合作为教学场景下的搭建与改装平台。

## 模块方案
磁吸模块安装于机体下方中心区域，尽量靠近机体重心。模块由磁吸组件、固定支架、紧固件和缓冲固定材料组成，目标是在不明显增加飞行阻力的前提下实现对金属目标物的吸附运输。

## 安装原则
- 不干涉桨叶旋转范围
- 不影响电池安装与拆卸
- 不遮挡飞控减震板和 GPS
- 尽量缩短载荷到机身中心的距离

## 关键设计点
- 支架安装孔位与机身底板匹配
- 磁吸部件与载荷接触面保持平整
- 结构件具备快速拆装能力
- 载货前后重心变化需记录并评估
