# 智能产品开发——磁吸运货无人机验证

## 项目简介
本项目采用现成无人机套件/材料包完成机体搭建，并在此基础上进行载货磁吸模块设计、安装与测试。项目目标是完成一套面向小型金属载荷的磁吸挂载验证方案，并通过结构改装、基础调试和飞行测试验证方案可行性。

## 项目目标
1. 完成无人机机体搭建与基础结构检查
2. 设计并安装磁吸载货模块
3. 验证空载与载货状态下的飞行稳定性
4. 完成磁吸抓取、运输与释放流程验证
5. 形成完整的文档、材料清单、测试记录和成员分工成果

## 核心方案
项目采用 F450 四旋翼平台作为基础机体，在机体底部加装磁吸挂载结构，实现对小型金属目标物的吸附与运输。系统由以下部分组成：

- 飞行平台：F450 机架、2212 电机、20A 电调、桨叶、电池与遥控链路
- 控制系统：Pixhawk 飞控、GPS、接收机、蜂鸣器、减震板
- 载货模块：磁吸组件、固定支架、紧固件、安装胶与扎带
- 验证任务：空载调试、载货飞行、吸附测试、落点与稳定性记录

## 建议实施流程
1. 按材料清单准备套件并完成机架、电机、电调与供电部分安装

3. 进行基础通电检查和空载调试
4. 设计磁吸模块，确定机身底部安装位置
5. 完成载货模块固定，复核重心与安全间隙
6. 开展空载、载货和磁吸任务测试
7. 汇总测试数据，完成答辩材料与项目总结
